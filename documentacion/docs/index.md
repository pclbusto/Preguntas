# Welcome to Preguntas Documentation

**Preguntas** is an interactive platform designed to **question and test student knowledge** through a variety of engagement methods.

Unlike traditional LMS platforms, **Preguntas** does not aim to provide theory, syllabus documentation, or explained content. Instead, it focuses on the active challenge: putting knowledge to the test.

## Project Vision

The core mission of **Preguntas** is to provide a rich, interactive environment where students are challenged through dynamic questioning. It is a tool for evaluation and reinforcement, not for theoretical exposition.

## Key Features

- **Interactive Questioning**: Multiple interaction types like drag-and-drop and text input.
- **Assessment Structure**: Organized into evaluation containers (Lessons) and sequences (Pages).
- **Gamified Feedback**: Immediate results, attempt tracking, and achievement medals.
- **Premium Interface**: A high-end glassmorphism design to keep students engaged during tests.
